# Banco de Dados de Usuários

Projeto desenvolvido durante o curso do SENAI, com o objetivo de praticar a criação de APIs utilizando Node.js, Express e banco de dados MySQL.

A aplicação permite realizar operações básicas de cadastro de usuários, seguindo o conceito de CRUD:

- Criar usuário
- Listar usuários
- Atualizar usuário
- Deletar usuário

---

## Tecnologias utilizadas

- Node.js
- Express
- MySQL
- MySQL2
- Dotenv
- JavaScript

---

## Objetivo do projeto

O objetivo deste projeto é aplicar os conhecimentos aprendidos em aula sobre desenvolvimento de APIs, integração com banco de dados e organização de rotas para manipulação de dados.

A API foi criada para gerenciar usuários em um banco de dados MySQL, permitindo cadastrar, consultar, editar e remover informações.
